# Alert Copy Rules

## 15m prealert

15m is only an entry-location reminder.

Allowed titles:
- `📍 BTC 15m 做空预警`
- `📍 BTC 15m 做多预警`

Meaning:
- not an official 1H confirmation;
- not a direction call by itself;
- just asks the trader to check whether there is a long/short entry location.

## 1H official alert

1H remains the main alert body.

Formal 1H titles may use stronger double emoji when confirmed:
- `📉 ... 📉`
- `📈 ... 📈`
- `✅ ...`

15m copy must not be mixed with 1H official confirmation copy.
